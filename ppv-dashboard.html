📚 Como configurar as APIs — passo a passo

+
▶ YouTube Data API v3
1

Acesse o Google Cloud Console

Vá para console.cloud.google.com e faça login com sua conta Google pessoal (@gmail.com).

2

Ative a YouTube Data API v3

No menu lateral, vá em APIs e Serviços > Biblioteca. Busque "YouTube Data API v3" e clique em Ativar.

3

Crie uma API Key

Vá em APIs e Serviços > Credenciais > Criar credenciais > Chave de API. Copie a chave gerada.

4

Encontre seu Channel ID

Acesse youtube.com/account_advanced. O ID começa com UC...

▶ Instagram via Meta Graph API
1

Crie um app no Meta for Developers

Acesse developers.facebook.com, clique em Meus Apps > Criar App. Escolha o tipo "Business".

2

Adicione Instagram Graph API

Na dashboard do app, encontre "Instagram Graph API" e clique em Configurar.

3

Gere um Access Token

Use o Graph API Explorer para gerar e converter para token de longa duração (60 dias).

4

Encontre seu Business Account ID

No Meta Business Suite, vá em Configurações > Contas do Instagram.

▶ TikTok for Business API
1

Acesse o TikTok for Developers

Vá para developers.tiktok.com e crie uma conta de desenvolvedor.

2

Crie um app

Em Manage Apps > Create app. Solicite acesso às permissões de analytics.

3

Obtenha o Access Token

Após aprovação, use o fluxo OAuth 2.0 documentado em developers.tiktok.com.

4

Alternativa mais rápida

Use o TikTok Studio e exporte os dados manualmente em CSV — a API TikTok para analytics pessoais tem restrições.
